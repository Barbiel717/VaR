§ getPtfVaR()

double VaRPtfCompute::getPtfVaR ( ) const

Compute dollar ptf's VaR.

$ VaRp = alpha * sDevp * P0 = alpha * sqrt(x' * Cov * x)

Computing ptf VaR in such way enables to derive analytical solution for iVaR, mVaR, cVaR.
